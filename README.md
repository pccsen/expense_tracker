# Expense Tracker CLI

Simple command-line application to manage your personal expenses.

## Project URL
https://roadmap.sh/projects/expense-tracker

## Features

- Add an expense
- Update an expense
- Delete an expense
- List all expenses
- Show summary of all expenses
- Show summary by month
